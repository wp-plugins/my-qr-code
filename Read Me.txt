== Description ==

You can use these in your sales pitch to your "Local Offline businesses", you can add the QR Code tag to ANY NEW or EXISTING post or page by clicking the 'Add Tag' widget to insert the short code that displays the 'QR Tag' where you want it to be displayed in your post, click to publish or update the post to finish the procedure.
My QR Code allows you to insert a QR Tag into any WordPress post or page, there are a number of different options that can be encoded into the QR Tag to make the user do as you want..., for instance you can make the user.

** Current add-ons **

* Browse to a Website
* Bookmark a Website
* Make a Phone Call
* Send an SMS
* Send an E-Mail
* Create a vCard
* Create a meCard
* Create a vCalendar Event
* Youtube URL for iPhone
* Tweet on Twitter
* Create Blackberry Messenger User
* WIFI Network for Android
* Free Formatted Text

If you have suggestions for a new add-on, feel free to email me at mrdimeh@eksit.co. Want regular updates? Follow me on Twitter!  http://twitter.com/mrdimeh


== Installation ==

1. Upload `My_QR_Code.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
